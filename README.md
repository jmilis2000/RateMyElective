# RateMyElective
RateMyProfessor clone to allow users to rate specific classes.

## Introduction
RateMyElective is inspired by RateMyProfessor. This website allows users to view the reviews of specific courses at their institution so that they can know what type of effort and instruction to expect. Users can also create an account so they can also leave reviews of courses that they have taken.

RateMyElective utilizes a Java backend with a HTML/CSS/Javascript frontend.
